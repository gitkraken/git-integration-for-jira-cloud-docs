Generate a quote for Git Integration for Jira Cloud by following these steps.

Click the [**Generate quote link**](https://www.atlassian.com/purchase/addon/com.xiplink.jira.git.jira_git_plugin.ondemand?purchaseMode=quote) to open the Cloud quote page:

![](/wp-content/uploads/gij-jira-cloud-quote-process.png)

1.  The page defaults to **Quote** mode.

2.  Select **Cloud** as the deployment type.

3.  Click **Add it** to install to your existing Jira instance.

4.  Follow the on-screen instructions to complete the process.


For more information on licensing, see [**Licensing Information, Migration and Transition**](/git-integration-for-jira-data-center/licensing-information-migration-and-transition-gij-self-managed).

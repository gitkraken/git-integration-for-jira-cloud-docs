Follow these steps to generate a quote for Git Integration for Jira Cloud.

## Generate a Quote

1. Open the [Generate quote page](https://www.atlassian.com/purchase/addon/com.xiplink.jira.git.jira_git_plugin.ondemand?purchaseMode=quote) on Atlassian Marketplace.

2. Verify the page displays **Quote** mode.

3. Select **Cloud** as the deployment type.

4. Click **Add it** to connect to your existing Jira instance.

5. Follow the on-screen instructions to complete your quote.

## Jira Server Platform Migration Planning

If you are planning to migrate to another platform, start planning your Jira Server to Cloud or Data Center migration now.

**Migration steps:**

1. Upgrade Git Integration for Jira Server app to the latest version.
2. Back up your data for migration.
3. Plan your migration path.

**Atlassian migration resources:**

- [Migrating Jira applications to another server](https://confluence.atlassian.com/adminjiraserver073/migrating-jira-applications-to-another-server-861253107.html)
- [Migrating from Jira Server to Jira Cloud](https://www.atlassian.com/migration/plan/cloud-guide)
- [Migrating from Jira Server to Jira Data Center](https://confluence.atlassian.com/enterprise/migrate-from-server-to-data-center-953127136.html)

## Browser Compatibility

Git Integration for Jira Cloud supports desktop browsers that Atlassian supports for Jira Cloud. Mobile browsers are not officially supported.

For more information, see [Atlassian Jira Cloud Supported Browsers](https://confluence.atlassian.com/cloud/supported-browsers-744721663.html).

## View Reindex Logs

1. Go to the Manage Git Repositories page.
2. Click the **Actions** icon.
3. Select **View Log** to view indexing logs.

### Internet Explorer 11 Support Timeline

With the release of the Edge browser in 2015, IE has not received major updates. Atlassian withdrew support for IE11 on all Jira products.

- **Atlassian Cloud products:** Support ended March 31, 2020
- **Atlassian server and data center products:** Last IE11-supported version released between September 2019 and March 2020

Git Integration for Jira Cloud provides this form for Jira administrators on the Advanced edition who need priority product support. Use this page when you are ready to submit an Advanced support case and already have the app details, entitlement information, and logs requested by the support guide.

**Requirements**

- Git Integration for Jira Cloud Advanced edition
- Jira Administrator access to the app configuration
- App details, entitlement number, and logs prepared before submission

## How to prepare your Advanced support request

See the [GIJ Advanced Support Guide](https://help.gitkraken.com/git-integration-for-jira-cloud/GIJ-Advanced-Support-Guide) for details on what information to include with your request and how to gather it.

&nbsp;

## Configure Automation for Jira triggers

Automate your workflows by connecting Git events to Jira actions. See [Git Integration + Automation for Jira](/git-integration-for-jira-cloud/git-integration-jira-automation-gij-cloud) for setup instructions.
